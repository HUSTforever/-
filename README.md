
# 🔮 奥秘塔罗 (Mystic Arcana) - 本地轻量版

> **"你的直觉，即是神谕。"**

「奥秘塔罗」是一款极具沉浸感的数字塔罗牌占卜应用。此版本为**纯前端本地版**，移除了对外部 AI 接口的依赖，所有的牌意解析均在本地即时完成。

## ✨ 功能特性

- **零配置，即点即用**：无需 API 密钥，无需科学上网，下载即可运行。
- **极致流畅体验**：移除网络请求，选牌到解读秒级跳转，仪式感不减。
- **交互式选牌动画**：包含平滑的横向滑动、卡片飞入动画以及 3D 物理翻牌效果。
- **隐私安全**：所有的占卜过程和问题均保存在浏览器本地，不会上传至任何服务器。
- **经典伟特牌组 (Rider-Waite)**：使用标准塔罗图像，内置核心牌意解析。

## 🛠️ 技术架构

- **核心框架**: React 19 (基于 ESM 模块化开发，无需 Node.js 环境构建)
- **样式方案**: Tailwind CSS (通过 CDN 引入)
- **图标/字体**: FontAwesome 6, Google Fonts
- **运行环境**: 支持任何现代浏览器 (Chrome, Edge, Safari, Firefox)

## 🚀 如何运行

本应用采用“零构建”设计，支持多种简易启动方式：

### 方式一：VS Code 预览（推荐）
1. 使用 VS Code 打开项目文件夹。
2. 安装 **Live Server** 扩展。
3. 右键点击 `index.html`，选择 **"Open with Live Server"**。

### 方式二：直接浏览器打开
虽然部分功能受限于浏览器的文件系统安全策略（CORS），但在大多数现代浏览器中，直接双击 `index.html` 亦可体验。

### 方式三：GitHub Pages 部署
1. 将代码推送至 GitHub 仓库。
2. 在 Settings > Pages 中开启部署。
3. 获得专属的在线占卜链接，随时随地访问。
